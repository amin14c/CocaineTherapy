# 🌱 New Life - تطبيق العلاج المساعد

> **تنبيه:** هذا المستودع خاص بمشروع تجربة سريرية مرخصة. الوصول إلى الكود مقيد للمصرح لهم فقط.

---

## 📋 نظرة عامة
**New Life** هو تطبيق أندرويد طبي مبتكر مصمم لدعم علاج إدمان الكوكايين ضمن بيئة مؤسسية منظمة. يعمل التطبيق كأداة تكميلية تحت الإشراف المباشر للأطباء النفسيين، ويهدف إلى تعزيز الالتزام العلاجي ومتابعة الحالة النفسية للمرضى.

## 🎯 الأهداف العلاجية
1. **المتابعة اليومية:** تسجيل المزاج، الرغبات، والأعراض الانسحابية.
2. **التمارين التفاعلية:** تقديم أنشطة علاجية مبرمجة وفق البروتوكول المعتمد.
3. **لوحة تحكم الطبيب:** تمكين الفريق الطبي من مراقبة المؤشرات الحيوية والنفسية.
4. **نظام الطوارئ:** تنبيه فوري للفريق الطبي في حالات الخطر أو الانتكاس.

---

## 🔒 الأمان والخصوصية (أولوية قصوى)

نظراً لحساسية البيانات الطبية وطبيعة التشخيص، يتبع التطبيق بروتوكولات أمان صارمة:

- **🚫 لا بيانات حقيقية في الكود:** يمنع منعاً باتاً تخزين أو رفع أي بيانات تعريفية للمرضى (PII) في هذا المستودع.
- **🔐 التشفير:** جميع البيانات المخزنة محلياً مشفرة باستخدام معايير `AES-256`.
- **🛡️ العزل:** فصل كامل بين طبقة العرض وطبقة البيانات والمنطق الطبي.
- **📵 الوضع المحلي:** التطبيق مصمم للعمل في بيئة معزولة (Offline-First) مع مزامنة آمنة مشفرة عند الحاجة.
- **🔒 المستودع الخاص:** هذا المستودع يجب أن يظل **Private** دائماً.

## 🏗️ هيكلية المشروع

```text
app/
├── data/          # طبقة البيانات (قواعد البيانات، التشفير، الـ Repositories)
├── domain/        # المنطق الطبي وقواعد العلاج (UseCases, Entities)
└── presentation/  # واجهة المستخدم (الشاشات، الـ ViewModels)
```

## 📂 المجلدات الهامة

## 🚀 للمطورين المصرح لهم

### المتطلبات
- Android Studio Hedgehog أو أحدث.
- JDK 17.
- صلاحيات وصول إلى المستودع الخاص.

### التحذيرات
- ⚠️ استخدم **بيانات وهمية فقط** (Dummy Data) أثناء التطوير.
- ⚠️ لا تقم بتعديل إعدادات الأمان أو إزالة التشفير.
- ⚠️ أي تعديل على المنطق العلاجي يجب أن يوافق عليه المشرف الطبي.
